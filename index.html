01 / Core

Single terminal for market data and scripted analysis.

Quantify Terminal is a local desktop app designed for Indian market workflows. It combines quotes, chart context, and command execution in one interface.

The runtime is compiled with C++20 and Qt6, and Python scripting is available for repeatable analysis tasks.
