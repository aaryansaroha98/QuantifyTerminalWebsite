04 / Build Command

Canonical source build flow.

setup.sh
$ git clone https://github.com/Quantify-Terminal/qt.git
$ cd qt
$ git checkout f1b3f18
$ cmake -B build -DCMAKE_BUILD_TYPE=Release
$ cmake --build build
